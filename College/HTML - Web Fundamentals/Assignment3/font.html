font

A shorthand property that sets several font properties in one declaration.

Possible values

  • style
  • variant
  • weight
  • size
  • line-height
  • font-family
p.a {font: 15px Arial, sans-serif;} p.b {font: italic small-caps bold 12px/30px Georgia, serif;}

font-style

The font-style property selects between normal, italic and oblique faces within a font family.

Possible values

  • normal
  • italic
  • oblique
p.a {font-style: normal;} p.b {font-style: italic;} p.c {font-style: oblique;}

font-variant

Will set an element’s text to small-caps

Possible values

  • normal (default)
  • small-caps
p.small {font-variant: small-caps;}

font-weight

Sets the boldness of a font.

Possible values

  • normal
  • bolder
  • bold
  • lighter
  • 100, 200, 300... 900
p.normal {font-weight: normal;} p.thick {font-weight: bold;} p.thicker {font-weight: 900;}

font-size

Sets the size of a font.

Possible values

  • percentage
  • length
  • larger
  • smaller
  • xx-small
  • x-small
  • small
  • medium (default)
  • large
  • x-large
  • xx-large
div.a {font-size: 15px;} div.b {font-size: large;} div.c {font-size: 150%;}

font-family

The font-family property specifies the font for an element.

The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.

There are two types of font family names:

Family-name - The name of a font-family, like “Times New Roman”, "courier", "arial", etc.

Generic-family - The name of a generic-family, like "serif", "sans-serif", "cursive", "fantasy", "monospace".

Start with the font you want, and always end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available.

Note: Separate each value with a comma. If a font name contains white-space, it must be quoted. Single quotes must be used when using the "style" attribute in HTML.

Possible values

  • font name
  • multiple font names separated by commas
p.a {font-family: "Times New Roman", Times, serif;} p.b {font-family: Arial, Helvetica, sans-serif;}
