letter-spacing

Specifies the spacing in-between letters. (Note: word-spacing is similar rule but specifies the spacing between words.)

Possible Values

  • length
  • normal (default)
h1 {letter-spacing: 3px;} h2 {letter-spacing: 2px;} h3 {letter-spacing: -1px;}
